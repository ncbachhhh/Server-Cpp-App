Test HTTP/1.0 Non-persistent Server

Nếu bạn thấy trang này qua trình duyệt/curl, nghĩa là GET ok.

Thử HEAD bằng curl:

curl -I http://127.0.0.1:8080/index.html

Link test

  • Link nội bộ (nếu file này là index.html): /index.html
  • Link gây 404 (để test response Not Found): /no_such_file.html
  • Thư mục (nếu bạn chưa hỗ trợ directory listing sẽ 403/404): /docs/

Form test (POST)

Nếu server chỉ hỗ trợ GET/HEAD, submit form sẽ ra 405/501.


Gợi ý test nhanh bằng curl

# GET
curl http://127.0.0.1:8080/index.html

# HEAD
curl -I http://127.0.0.1:8080/index.html

# Test 404
curl -i http://127.0.0.1:8080/no_such_file.html

# Test method không hỗ trợ
curl -i -X POST http://127.0.0.1:8080/index.html
